## 🚀 Projetos

### 🏃 HexRun
App mobile de corrida gamificado onde a cidade é dividida em tiles hexagonais.  
Ao correr, o usuário conquista território, compete com outros corredores e forma crews.

**Principais tecnologias:**

- React Native + Expo
- TypeScript
- Mapbox
- Uber H3 (geospatial indexing)
- Supabase (PostgreSQL + PostGIS)
- Zustand
- Reanimated

O projeto combina **geolocalização, gamificação e dados espaciais** para transformar corrida em um jogo multiplayer no mundo real.
