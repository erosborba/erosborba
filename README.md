# 🏃 HexRun

**HexRun** é um aplicativo de corrida gamificado onde a cidade é dividida em tiles hexagonais.

Ao correr, o usuário:

- conquista territórios
- compete com outros corredores
- forma crews
- completa objetivos semanais
- compartilha mapas de progresso

A ideia é transformar corrida em um **jogo multiplayer no mundo real**.
